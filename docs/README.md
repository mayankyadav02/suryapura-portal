# 📖 Project Overview

**Suryapura Gram Vikas Portal** is a fictional rural development platform created using **React + Vite**.

The project demonstrates how a modern village can use technology to improve governance, education, agriculture, healthcare, transparency, and citizen engagement through a premium digital experience.

The portal focuses on delivering a clean, responsive, bilingual, accessible, and modern user interface while following scalable React architecture and reusable component design.

# 🛠 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/suryapura-portal.git
```

Go to the project folder:

```bash
cd suryapura-portal
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

---

# 📦 Build

Production Build

```bash
npm run build
```

Preview Production

```bash
npm run preview
```
